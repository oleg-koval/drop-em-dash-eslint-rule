# Contributing

Thanks for helping improve `eslint-plugin-drop-em-dash`.

## Development

1. Clone the repository and install dependencies:

   ```bash
   git clone https://github.com/oleg-koval/drop-em-dash-eslint-rule.git
   cd drop-em-dash-eslint-rule
   npm install
   ```

2. Run the test suite and lint checks:

   ```bash
   npm test
   npm run lint
   ```

3. Make your change with a focused commit message (see [Conventional Commits](https://www.conventionalcommits.org/) if you like that style).

## Pull requests

- Keep changes small and scoped to one concern.
- Add or update tests for any behavior change.
- Ensure `npm test` and `npm run lint` pass locally before opening a PR.

## Code of conduct

This project follows the [Contributor Covenant](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.
